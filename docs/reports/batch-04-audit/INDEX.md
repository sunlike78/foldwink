## Self-flagged concerns for the validator

1. **0216 Coast to coast — "Marine Iguana"** in Saltwater Reptiles: this is a real Galapagos species, but most readers will picture a land iguana. If validator flags ambiguity, swap for a different saltwater reptile (e.g. collapse the group to three groups of iguana-free content — but saltwater reptiles only has four living species globally, so options are thin).
2. **0218 Concert night — "Mute" / "Tuner"**: both items are generic English words in other contexts. The "Musician's Gear" label should pin them; if the group still reads as ambiguous on first glance, swap for "Case" and "Strap".
3. **0222 Wind has a name — "Trade Wind"**: the only item in Named Regional Winds that is not a proper single-word name. It reads differently from Sirocco / Chinook / Mistral. Verify this does not trigger a "doesn't match siblings" flag.
4. **0224 Pan across — first-word uniformity**: all four group labels start with "Pan" by design (same as approved puzzle-0121 "Flat"). Confirm this precedent still holds.
5. **0226 Salt and rope — "Bosun" / "Bowsprit"**: slightly specialist nautical terms. Medium tier tolerates this, but watch that the revealHint "Hull" / "Sailors" provides enough pull.
6. **0228 Anatomy of a song — continuous gradient risk**: Tempo (Allegro → Largo) and Dynamics (Fortissimo → Pianissimo) are each internal gradients. They do not cross between groups, but if the validator reads "tempo vs dynamics" as one gradient from "how a piece sounds", flag it early.
7. **0230 Feeling blue — colour attribute fairness**: batch-03 warm-palette (0201) was rejected for continuous-gradient failure. This puzzle is deliberately a mixed-attribute puzzle (shades / birds / foods / idioms), not a shades-gradient puzzle. But "Indigo" vs "Periwinkle" still live on a gradient. If fairness fails, drop the experiment.
8. **0211 Jewellery box — Ring Types group**: three of four items ("Wedding Ring", "Pinky Ring", "Thumb Ring") contain the word "Ring". Label "Ring Types" may telegraph the group too clearly and reduce puzzle challenge. If validator prefers tighter item variety, replace two items with "Signet" sibling terms.
9. **Cross-puzzle item reuse within batch-04** (all benign, different categories):
   - "Chinook" — 0208 (helicopter) + 0222 (wind). Different categories.
   - "Oxford" — 0209 (shirt) + 0210 (shoe). Different categories.
   - "Oven Mitt" — 0215 (kitchen textile) + 0225 (glove style). Different categories.
