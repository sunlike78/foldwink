## Self-flagged concerns for the validator

1. **0233 Knife Cuts group** — "Dice" collides with my own 0250 "Polyhedral Dice" label token and with pool-hard-002 ("Fold" in poker). Cross-puzzle item reuse in different categories (verb vs noun vs game concept). Validator should verify within-easy context pin holds.
2. **0235 Dragon and dumpling** — "Cherry Blossom" appears in pool-0204 (Spring Blooms) and batch-04 0214 (seasons). Shared item, different puzzle frame (symbol vs bloom). Acceptable cross-puzzle reuse expected.
3. **0236 Round table** — Arthurian legend is uncommon as an easy puzzle; Tintagel and Glastonbury Tor lean specialist. If validator flags, swap Tintagel → "The Forest" and Glastonbury Tor → "Castle", though these are weaker.
4. **0238 Trick or treat** — "Witch" overlaps with pool-0194 Fairy Tale Villain; "Skeleton" with pool-0131 (winter sports) and 0180 (body frame). Cross-puzzle item reuse, Halloween-frame label pins in this puzzle.
5. **0242 In the cellar** — Wine varieties Merlot/Cabernet/Chardonnay are common household terms; Riesling/Pinot Grigio/Sauvignon Blanc slightly more specialist but still broadly recognised. Napa Valley "Napa" word overlaps with pool-0086 Cabbage Family (Napa cabbage) but my item is "Napa Valley" distinct.
6. **0244 Read and write — "Arabic"** — is both an alphabet and a language. Pool-0013 has Arabic as language item. Within 0244's context the label "World Alphabets" pins, and no Languages group exists in 0244. Still, worth flagging.
7. **0246 Bottled scents — Spicy Notes** — Cinnamon/Clove/Cardamom/Pepper is a valid fragrance family and a valid Spices group. Pool has Spices (0005, 0113). Medium-tier false trail is intentional here; revealHints pin. If validator considers this unfair, Spicy Notes can swap Pepper → Nutmeg (but same risk).
8. **0247 Quiet feelings — Pride group "Honor"** — "Honor" also appears in 0236 Knightly Themes. Cross-batch reuse across different puzzles (emotion vs knightly theme); benign but noted.
9. **0248 Greek heroes** — Priam/Agamemnon lean specialist (Iliad characters). Medium tolerates this, but if validator rejects, swap "Priam" → "Tantalus" (Midas/Tantalus/Oedipus/Agamemnon is tighter mortal-king-with-tragedy axis; Jason being in Heroes group disambiguates).
10. **0249 Take the shot — Photo shot "Portrait"** — pool-0142 has Photo Styles group with "Portrait", and pool-0180 Picture frame has "Canvas". Portrait is generic enough to exist in several puzzles. Within this medium the revealHint "Camera" pins the photo sense.
11. **0251 Pastry school — Meringue Types** — items "French / Italian / Swiss / Japanese" look like nationalities in isolation. Medium disambiguation is the design intent; revealHint "Whipped" is the pin. Flagging as designer concern.
12. **0252 Road to space — "Steam Train"** — item appears in pool-0082 Train Types and batch-04 0206 All aboard (Bullet/Freight/Steam/Monorail). Cross-puzzle reuse, different framing (transport era vs train classification). Acceptable expected.
13. **0254 Take the plunge** — "Mountaineering" in Climbing Sports is close to "Mountain Biking" in Wheeled Extreme — both start with "Mountain" but the label and remaining item set pin clearly. If validator considers the "Mountain" surface-prefix a signal-inversion issue, swap Mountain Biking → Off-Road Cycling.
