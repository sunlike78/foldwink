# Foldwink — posting notes

Короткие практические заметки. Не «маркетинговая стратегия» — только то, что реально влияет на решение "какой ролик куда".

---

## Какие 3 ролика публиковать первыми

1. **`challenge_v1.mp4`** — «Почему большинство ошибается здесь с первой попытки?»
   Tension-хук в первом кадре + почти-провал в середине + чистое решение. Высокий потолок по комментам («угадал / не угадал»).
2. **`satisfying_v2.mp4`** — «Чистый кайф для мозга».
   Самый короткий (11с), идеальный loop, подходит любой аудитории. Низкий бар входа.
3. **`challenge_v3.mp4`** — «Сможешь решить без единой ошибки?»
   Прямой вопрос к зрителю — провоцирует response-видео и дуэт-ответы.

Остальные 3 (satisfying_v1, satisfying_v3, challenge_v2) — в ротацию через 2–3 дня.

---

## Платформенное разделение

| Ролик           | TikTok      | Shorts     | Reels  |
| --------------- | ----------- | ---------- | ------ |
| `satisfying_v1` | ✅ основной | ✅         | хорошо |
| `satisfying_v2` | ✅ loop     | ✅ primary | ✅     |
| `satisfying_v3` | средне      | ✅         | ✅     |
| `challenge_v1`  | ✅ primary  | хорошо     | ✅     |
| `challenge_v2`  | ✅ reveal   | ✅         | средне |
| `challenge_v3`  | ✅ CTA      | ✅ CTA     | ✅     |

**TikTok** — сильнее отрабатывают tension-хуки и reveal-ролики (более высокий completion ценится).
**Shorts** — любят короткие loop-friendly (10–12с) и явный CTA в конце.
**Reels** — ROI сильно зависит от caption; не экономить на тексте.

---

## Hook → ожидаемый эффект

| Hook                            | Сильнее в          | Почему                                    |
| ------------------------------- | ------------------ | ----------------------------------------- |
| «POV 5 минут»                   | retention          | ставит сцену без требования действия      |
| «Чистый кайф для мозга»         | rewatch            | короткий, безопасный loop                 |
| «Новая головоломка каждый день» | saves              | пользователь сохраняет «вернуться завтра» |
| «Почему большинство ошибается»  | comments           | прямой challenge triggers replies         |
| «Я специально ошибся»           | rewatch + comments | сюжетный твист → пересмотр                |
| «Сможешь без единой ошибки»     | comments + clicks  | closed yes/no question                    |

---

## Что не делать

- Не ставить критический текст в зоне `y > 85%` — закроется UI TikTok/Shorts.
- Не выкладывать оба challenge-видео в один день — конкурируют за reach.
- Не использовать одинаковые хэштеги все 6 раз — ротируй (`hashtags.txt`).
- Не добавлять звук из trending songs без проверки лицензии — игра тихая, overlay-musik опциональна.
