## Вердикт
- Да, пересечение несправедливое.
- `Овца` и `Баран` семантически связаны как самка и самец одного вида, а разделение между `Домашний скот` и `Мелкий скот` не даёт игроку надёжного правила, почему они должны быть в разных группах. Дополнительно сами категории `Домашний скот` и `Мелкий скот` пересекаются: мелкий скот всё равно является домашним скотом.

## Предложенный фикс
### Вариант 1 (минимальный)
Минимально сохранить текущий набор животных: перенести `Овца` в группу с мелким скотом, `Осёл` — в более крупные фермерские животные, а label второй группы уточнить так, чтобы он не пересекался с «мелким скотом».

```json
"groups": [
  {
    "id": "group_a",
    "label": "Домашние питомцы",
    "items": ["Кошка", "Собака", "Хомяк", "Попугай"]
  },
  {
    "id": "group_b",
    "label": "Крупные фермерские животные",
    "items": ["Корова", "Лошадь", "Свинья", "Осёл"]
  },
  {
    "id": "group_c",
    "label": "Домашняя птица",
    "items": ["Курица", "Утка", "Гусь", "Индейка"]
  },
  {
    "id": "group_d",
    "label": "Мелкие фермерские животные",
    "items": ["Коза", "Кролик", "Баран", "Овца"]
  }
]
```

### Вариант 2 (если нужен альтернативный)
Если хочется убрать пару `Овца`/`Баран` вообще, можно заменить `Баран` на животное, которое не является парой к `Овца`. Например, `Нутрия` сохраняет фермерско-домашний контекст, но чуть снижает культурную доступность для `easy`, поэтому этот вариант хуже первого.

```json
"groups": [
  {
    "id": "group_a",
    "label": "Домашние питомцы",
    "items": ["Кошка", "Собака", "Хомяк", "Попугай"]
  },
  {
    "id": "group_b",
    "label": "Домашний скот",
    "items": ["Корова", "Лошадь", "Свинья", "Овца"]
  },
  {
    "id": "group_c",
    "label": "Домашняя птица",
    "items": ["Курица", "Утка", "Гусь", "Индейка"]
  },
  {
    "id": "group_d",
    "label": "Мелкие фермерские животные",
    "items": ["Коза", "Кролик", "Нутрия", "Осёл"]
  }
]
```

## Сложность
easy, если применить вариант 1. После уточнения `Крупные фермерские животные` / `Мелкие фермерские животные` группы становятся достаточно прямыми для лёгкого пазла: питомцы, птица, крупные фермерские, мелкие фермерские. В текущем виде я бы не поднимал до `medium`, а считал бы пазл некорректным для публикации из-за нечестной двусмысленности.

## Дополнительные пары для seed
- овца / баран — самка и самец одного вида; сильный риск ложной пары
- индейка / индюк — самка и самец одного вида, особенно опасно рядом с домашней птицей
- курица / петух — самка и самец одного вида; частая бытовая пара
- корова / бык — самка и самец одного вида; риск при группах про скот
- лошадь / жеребёнок — взрослое животное и детёныш; риск при группах «животные» и «детёныши»
